The Problem Space

  • Lack of Structure: Coachees struggle with independent task completion.
  • Generic Tools: Specific, guided instruction is missing in current apps.
  • Disconnected Systems: Coaches lack a centralized tracking hub.

Result: Confusion, inconsistency, and poor accountability.

Our Tech Solution

A cloud-app providing structured, guided task management.

  • Frontend: React Native / Expo
  • Authentication: Clerk (Role-based access)
  • Database: Firebase Firestore (NoSQL real-time sync)
  • Storage: Firebase Storage (Cloud image hosting)

Why This Architecture?

  • Soundness: Serverless Architecture
  • Relevance: Real-time Tracking
  • Novelty: Granular Media
