# TaskMate Prototype (Phase 1)

This is the initial prototype for the Messenger interface. It uses mock data to demonstrate the Core Loop.

## Features
- **Dashboard**: View assigned tasks.
- **Task Guidance**: Step-by-step instructions with Text-to-Speech (TTS).
- **Completion**: Celebration screen when task is done.

## How to Run
1. Install dependencies (if you haven't):
   ```bash
   npm install
   ```
2. Start the Metro bundler:
   ```bash
   npx expo start
   ```
3. Scan the QR code with your phone (using Expo Go app) or press `a` for Android Emulator / `i` for iOS Simulator.

## Tech Stack
- React Native (Expo)
- NativeWind (Tailwind CSS)
- React Navigation
- Expo Speech
